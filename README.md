# ⚡ Next.js Learning Project

This repository contains my **Next.js learning project** built while following the tutorial series by **CodeWithHarry**.

Next.js is a powerful **React framework** that enables developers to build fast and scalable web applications with features like **server-side rendering, static site generation, and API routes**.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```id="sn5c7k"
git clone https://github.com/your-username/nextjs-project.git
```

### 2️⃣ Navigate to the Project Folder

```id="n5eh1k"
cd nextjs-project
```

### 3️⃣ Install Dependencies

```id="n2p9p7"
npm install
```

### 4️⃣ Run the Development Server

```id="o9j8dl"
npm run dev
```

Now open your browser and go to:

```id="o5ab2y"
http://localhost:3000
```
